# AI Factory SPA Dashboard

Premium vanilla HTML/CSS/JS single page dashboard demo.

## Routes

- `#/home` — hero, feature cards, overview metrics
- `#/dashboard` — telemetry, animated counters, canvas chart, event logs
- `#/calculator` — model selector, token sliders, cost total, ratio bars
- `#/workflow` — interactive SVG workflow, details, FAQ/docs, copy snippets
- `#/contact` — validated contact form, errors, spinner, toast
